# Test Page Specification

Create a modern, professional homepage for a fictional SaaS company called "TaskFlow".

## Company Overview
TaskFlow is a project management tool for remote teams. Target audience: small to medium businesses (10-50 employees).

## Page Requirements

### Hero Section
- Compelling headline about streamlining team collaboration
- Subheadline explaining the core value proposition
- Primary CTA: "Start Free Trial"
- Secondary CTA: "Watch Demo"
- Hero image or illustration

### Features Section
- 3-4 key features with icons
- Each feature should have a title and brief description
- Focus on: Real-time collaboration, Task automation, Analytics dashboard, Mobile app

### Social Proof
- Customer testimonials (2-3)
- Company logos of fictional clients
- Statistics (users, tasks completed, etc.)

### Pricing Teaser
- Brief mention of pricing tiers
- CTA to view full pricing

### Footer
- Company links (About, Careers, Blog)
- Product links (Features, Pricing, Security)
- Social media icons
- Copyright notice

## Design Requirements
- Modern, clean design
- Professional color scheme (blues/greens suggested)
- Mobile responsive
- Accessible (WCAG 2.1 AA)
- Fast loading
- Clear visual hierarchy

## Technical Requirements
- Semantic HTML5
- Modern CSS (flexbox/grid)
- Minimal JavaScript (smooth scroll, mobile menu)
- SEO optimized (meta tags, headings, alt text)
